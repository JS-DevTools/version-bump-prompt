# Version-bump

Update project version and create tag easily

## Install

	npm install -g version-bump

## Usage

	bump [options]

 * `--patch`, `--minor`, `--major` - Increase specific version
 * `--no-tags` - Do not create git tag
 * `--push` - Push to remote repo
